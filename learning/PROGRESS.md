## 使用规则

1. 每完成一个阶段，可以在自己的练习分支更新一行记录。
2. 不要记录密码、令牌、邮箱验证码或公司敏感信息。
3. 提交前使用 `git diff --staged` 检查实际内容。
4. 学习提交使用 `docs:` 类型，例如 `docs: record Git learning checkpoint`。
